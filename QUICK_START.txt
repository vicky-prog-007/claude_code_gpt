‚ö° 3-MINUTE SETUP:
================================================================================

STEP 1: Configure Azure Credentials
------------------------------------
1. Copy config.example.bat to config.bat
2. Edit config.bat and fill in your Azure credentials:
   - AZURE_OPENAI_ENDPOINT
   - AZURE_OPENAI_API_KEY
   - AZURE_OPENAI_DEPLOYMENT (usually "gpt-5")
   - AZURE_OPENAI_API_VERSION (use "2025-01-01-preview")

STEP 2: Install Dependencies
-----------------------------
Open Command Prompt and run:
   pip install fastapi uvicorn httpx

STEP 3: Configure VS Code
--------------------------
1. Open VS Code Settings (Ctrl+,)
2. Search for "anthropic"
3. Set these values:
   - Anthropic: Base Url = http://127.0.0.1:5006
   - Anthropic: Api Key = dummy

STEP 4: Start the Proxy
------------------------
Double-click: Start-ClaudeProxy.bat

You should see:
   ========================================
   Proxy: http://127.0.0.1:5006
   Model: Azure GPT-5
   Tools: ENABLED (Read, Write, Bash, etc.)
   ========================================

STEP 5: Use Claude Code!
-------------------------
Open Claude Code extension in VS Code and start coding!
